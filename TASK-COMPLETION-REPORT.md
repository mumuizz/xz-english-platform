## 🛠️ 技术实现

### 后端 API

**单词接口** (`/api/words`):
- `GET /words` - 获取单词列表
- `GET /words/due` - 获取待复习单词
- `POST /words` - 添加单词
- `POST /words/import-sample` - 导入词库
- `PUT /words/:id/review` - 复习单词
- `DELETE /words/:id` - 删除单词

**文章接口** (`/api/articles`):
- `GET /articles` - 获取文章列表
- `GET /articles/:id` - 获取单篇文章
- `POST /articles/import-people-daily` - 导入文章

**听力接口** (`/api/listening`):
- `GET /listening` - 获取听力列表
- `GET /listening/:id` - 获取单个听力
- `POST /listening/import` - 导入听力材料

- ✅ `Ebbinghaus.tsx` - 单词记忆页面（已更新导入提示）

## 💡 使用建议

### 首次使用流程

1. **注册/登录** → 创建账号
2. **导入词库** → 单词记忆页面点击"📥 导入词库"
3. **导入文章** → 阅读页面点击"📥 导入人民日报文章"
4. **导入听力** → 听力页面点击"📥 导入听力材料"
5. **开始学习** → 选择词库，开始复习

### 日常学习流程

1. 📚 **单词记忆** - 使用艾宾浩斯记忆法复习
2. 📰 **阅读文章** - 每天读 1-2 篇文章
3. 🎧 **听力训练** - 精听 1 个听力材料
4. ✅ **每日打卡** - 记录学习进度
